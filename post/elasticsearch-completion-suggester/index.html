Elasticsearch自动补齐建议-completion suggester

发表于 2017-03-17 | 分类于 ElasticSearch |

mapping 添加测试数据 DSL 结果 代码

1.mapping

curl -XPUT 192.168.0.1:9200/person -d'  //新建一个persion的索引
{
   "mappings": {
      "person": { //这个是_type
         "properties": {
            "name": {
               "type": "string"
            }
            "tag_suggest": {
               "type": "completion",  //设置为completion才能被suggest捕获
               "index_analyzer": "ik",
               "search_analyzer": "ik",
               "payloads": false
            }
         }
      }
   }
}'

2 .添加测试数据

curl -XPUT  192.168.2.20:9200/person/person/1 -d'
{
"name": [
     "david",
    "jacky"
 ],
"tag_suggest": {
    "input": [
     "david",
     "jacky"
  ]
  }
}'
curl -XPUT 192.168.0.1:9200/person/person/1 -d'
{
"name": [
    "andy",
    "jackson"
],
 "tag_suggest": {
"input": [
     "andy",
    "jackson"
  ]
}
}'

3.DSL

curl -XPOST 192.168.0.1:9200/person/_suggest -d'
{
 "person_suggest":{
      "text":"jack",
     "completion": {
          "field" : "tag_suggest"
    }
}
}'
