Elasticsearch自动补齐建议-completion suggester

Aug 08, 2016 in ElasticSearch

Table of Contents

  1. 1.mapping
  2. 2 .添加测试数据
  3. 3.DSL
  4. 4.结果
  5. 5.代码

1.mapping

curl -XPUT 192.168.0.1:9200/person -d'  //新建一个persion的索引
{
   "mappings": {
      "person": { //这个是_type
         "properties": {
            "name": {
               "type": "string"
            }
            "tag_suggest": {
               "type": "completion",  //设置为completion才能被suggest捕获
               "index_analyzer": "ik",
               "search_analyzer": "ik",
               "payloads": false
            }
         }
      }
   }
}'

 "tag_suggest": {

3.DSL

curl -XPOST 192.168.0.1:9200/person/_suggest -d'
{
 "person_suggest":{
      "text":"jack",
     "completion": {
          "field" : "tag_suggest"
    }
}
}'
